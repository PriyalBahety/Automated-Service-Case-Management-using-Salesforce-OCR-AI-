 Automated Service Case Management System
📖 Overview
This project automates customer complaint handling using Salesforce. It converts emails into service cases and processes them using OCR (simulated) and AI concepts for classification, prioritization, and routing. The system reduces manual effort and improves efficiency.

🚀 Features
Email-to-Case automation

OCR-based data handling (simulated)

AI-based complaint classification

Automatic priority assignment

Case routing to departments

Case tracking system

🛠️ Technologies Used
Salesforce CRM

OCR (concept-based)

AI/ML concepts

⚙️ System Flow
Email → Case Creation → Data Processing → AI Classification → Routing → Resolution
